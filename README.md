# EDA Airbnb NYC 2019

## 📊 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Airbnb NYC 2019 dataset to understand pricing, availability, and neighborhood trends.

## 🛠 Tools Used

* Python
* Pandas
* NumPy
* Seaborn
* Matplotlib

## 📌 Key Analysis

* Univariate Analysis (price, availability_365)
* Bivariate Analysis (price vs room_type, location)
* Multivariate Analysis (heatmaps, scatter plots)
* Outlier Detection & Treatment (price using IQR)
* Missing Value Handling

## 📈 Key Insights

* Entire homes have higher prices compared to shared rooms
* Manhattan has the highest average prices
* Availability varies significantly across neighborhoods

## 📂 Dataset

Airbnb NYC 2019 dataset from Kaggle
